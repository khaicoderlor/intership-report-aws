### Kết quả đạt được tuần 1:

* **Nền tảng AWS:**
  * Tạo thành công tài khoản AWS với billing alerts và MFA
  * Hiểu về hạ tầng toàn cầu AWS và chiến lược chọn region
  * Quen thuộc với AWS Management Console

* **Bảo mật Ưu tiên:**
  * Thực hiện IAM best practices: tạo admin user, bảo vệ root account
  * Cấu hình password policy và MFA cho tất cả accounts
  * Hiểu nguyên tắc least privilege

* **Version Control:**
  * Thiết lập Git và GitHub account
  * Học Git Flow branching strategy cho team collaboration
  * Tạo repository đầu tiên với README documentation đầy đủ

* **Kỹ năng Networking:**
  * Thiết kế và triển khai custom VPC với CIDR planning
  * Hiểu sự khác biệt giữa public và private subnets
  * Cấu hình Internet Gateway cho public internet access

* **Compute Cơ bản:**
  * Khởi chạy và kết nối thành công tới EC2 instance
  * Hiểu các loại instance types và use cases
  * Thực hành SSH key management và cấu hình Security Group
